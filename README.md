# database-app-practice

## 概要
COACHTECH 教材 Tutorial 9-3「データベース操作ハンズオン演習」で作成した成果物です。
- マイグレーションファイルでテーブル定義する仕組み（Schema::create、各種カラム型）
- シーダーでダミーデータを一括投入する流れ
- クエリビルダー（DB::table()）で一覧データを取得する方法

## 使用技術
- PHP 8.x
- Laravel 10.x
- MySQL
- マイグレーション、シーダー
（**他に使ったものがあれば追記してください**）

## 学んだこと
- 【ステージ A：土台（データベース）を作る】
 ├── Step 1. マイグレーションを作る (設計図を用意する) 
 ├── Step 2. マイグレーションを実行する (データベースに実際の表を作る)
 └── Step 3. シーダーを実行する (テスト用のデータを表に書き込む)
        ▼
【ステージ B：裏側の仕組み（案内板とデータ取得）を作る】
 ├── Step 4. コントローラーを作る (データを取ってくる司令塔を用意する)
 └── Step 5. ルーティングを決める (「このURLに来たら、司令塔を動かして」と設定する)
        ▼
【ステージ C：表側の見た目（画面）を作る】
 └── Step 6. Bladeファイルを作る (取ってきたデータを綺麗に並べて、ユーザーに見せる)
